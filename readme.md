# BuyOrBye

BuyOrBye is an ASP.NET Core MVC web application built with C# that allows users to browse products and share reviews and ratings.

The purpose of this application is to help users decide whether a product is worth buying by providing customer feedback and product information.

PROJECT LINK
http://buyorbye.somee.com/

## Models

### Product
Stores product information such as:
- Name
- Brand
- Price

### Review
Stores customer review information such as:
- Comment
- Rating
- Review Date

A Product can have many Reviews.

## Technologies Used

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Visual Studio
- GitHub

## Bonus Feature
Implemented sorting functionality using LINQ OrderBy and OrderByDescending in ASP.NET Core MVC.
